

iPad外接键盘终极快捷键指南

📝 文本编辑快捷键

类别 功能说明 快捷键 已验证
基础操作 全选文本 Command (⌘) + A  ☐
 剪切选中的文本 Command (⌘) + X  ☐
 拷贝选中的文本 Command (⌘) + C  ☐
 粘贴剪切板内容 Command (⌘) + V  ☐
 撤销上一步操作 Command (⌘) + Z  ☐
格式调整 将文本设为粗体 Command (⌘) + B  ☐
 将文本设为斜体 Command (⌘) + I  ☐
查找替换 查找（打开查找窗口） Command (⌘) + F  ☐
 查找下一个 Command (⌘) + G  ☐
 查找上一个 Shift + Command (⌘) + G  ☐

⚙️ 系统操作快捷键

类别 功能说明 快捷键 已验证
界面导航 返回主屏幕 Command (⌘) + H  ☐
 显示或隐藏搜索栏 Command (⌘) + 空格键  ☐
 显示或隐藏程序坞 Option (⌥) + Command (⌘) + D  ☐
多任务 在打开的App间切换 Command (⌘) + Tab  ☐
屏幕截图 拍摄截屏 Command (⌘) + Shift + 3  ☐
 拍摄截屏并立即打开标记 Command (⌘) + Shift + 4  ☐
快速设置 快速锁定屏幕 Control + Command (⌘) + Q  ☐

📱 应用内与通用快捷操作

类别 功能说明 快捷键 已验证
通用功能 添加批注（如在Pages中） Shift + Command (⌘) + K  ☐
 显示或隐藏标尺（如在Pages中） Command (⌘) + R  ☐
页面导航 向上滚动一页 Fn + 上箭头 或 Page Up  ☐
 向下滚动一页 Fn + 下箭头 或 Page Down  ☐
插入操作 插入分页符（如在Pages中） Command (⌘) + Return 或 Command (⌘) + Enter  ☐

🔍 查看与自定义快捷键

类别 功能说明 操作 已验证
查看快捷键 查看当前App的所有可用快捷键 按住 `Command (⌘)` 键不放  ☐
自定义快捷键 创建属于自己的快捷键 前往 "设置" > "辅助功能" > "键盘" > "全键盘控制"，开启后即可自定义命令。 ☐

· 修饰键自定义：你可以更改外接键盘上修饰键（如 Command、Option、Caps Lock）的功能。前往 "设置" > "通用" > "键盘" > "实体键盘" > "修饰键" 进行设置。
· PC键盘习惯：如果你更熟悉PC键盘，可以记住iPad上的 Command (⌘) 键功能类似于PC上的 Control (Ctrl) 键。
